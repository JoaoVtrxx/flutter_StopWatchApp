# Cronômetro

Projeto de um Cronômetro em Flutter.

## Funcionalidades
- Iniciar o Cronômetro;
- Pausar o Cronômetro;
- Resetar o Cronômetro;
- Salvar volta/recorde de Tempo em um lista scrollável;
